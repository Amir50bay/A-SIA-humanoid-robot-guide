# A-SIA - Humanoid Robotic Guide Prototype

> Servo-driven facial and head motion · Voice playback · Wheeled mobile base · Arduino Mega 2560

---

## Overview

A-SIA is a humanoid robotic guide prototype developed for interactive public demonstration. The system combines servo-actuated facial expression, head motion, and audio playback into a single embedded platform built around the Arduino Mega 2560.

The prototype was demonstrated at the **NIS Turkistan Hackathon**, where it operated as a stationary interactive guide with live facial motion and triggered voice phrases.

This repository is a structured restoration of the original project based on preserved code, schematics, FLProg project files, and photographs of the physical prototype.

---

## Key Features

- **7-channel servo system** driving head yaw, head pitch, mouth, eye axes, and bilateral facial axes
- **Wired analog control** via potentiometers on A0–A6 mapped directly to servo angles
- **DFPlayer Mini audio subsystem** for prerecorded guide phrase playback
- **Three-track audio trigger** via RemoteXY interface (K1 / K2 / K3)
- **RemoteXY + ESP8266** for supplementary wireless UI overlay
- **RTC DS1302** for real-time clock display
- **I²C LCD (16×2)** showing time and environmental readings
- **Custom oval mobile base** as a load-bearing rolling platform for demonstration

## Restored Servo Map

All seven servo channels are preserved from the original FLProg-generated code. Pin assignments are confirmed from both the archived sketch and the hand-drawn wiring diagram.

| # | Arduino Pin | Analog Input | Role |
|---|---|---|---|
| SM1 | D2 | A0 | Head Yaw |
| SM2 | D3 | A1 | Head Pitch |
| SM3 | D4 | A2 | Mouth |
| SM4 | D5 | A3 | Eyes Horizontal |
| SM5 | D6 | A4 | Eyes Vertical |
| SM6 | D7 | A5 | Left Facial Axis |
| SM7 | D8 | A6 | Right Facial Axis |

Each servo angle is computed as:
```cpp
servo.write(map(analogRead(Ax), 0, 1023, 0, 180));
```
Control was fully wired - potentiometers physically attached to the operator panel.

## System Architecture

```
┌─────────────────────────────────────────────────┐
│               Arduino Mega 2560                 │
│                                                 │
│  A0–A6 ──► Servo angle mapping (SM1–SM7)        │
│  D2–D8 ──► Servo signal output                  │
│  D10/D11 ─► DFPlayer Mini (SoftwareSerial)      │
│  Serial1  ─► ESP8266 RemoteXY (K1/K2/K3)        │
│  I²C      ─► LCD 16×2 (0x27)                    │
│  D23/25/27 ► DS1302 RTC                         │
│  D28      ─► DHT sensor                         │
└─────────────────────────────────────────────────┘
         │                        │
   Head / Face              Audio Output
   7× SG90 servos           DFPlayer Mini
   wired potentiometers      microSD card
                             3 prerecorded tracks
```

The AI guide layer was external - a conversational persona was operated separately and used to script the guide interaction. The embedded system handled all motion and audio; no onboard inference or LLM API was present in the firmware.

## Preserved Code Evidence

The archived FLProg-generated sketch (`archived/legacy_flprog_code.ino`) confirms:

- All 7 servos attached on D2–D8
- Analog inputs A0–A6 wired to servo angle mapping
- `SoftwareSerial mySerial(10, 11)` for DFPlayer communication
- `dfPlayerMega.start_track(1, x)` calls for tracks 1, 2, and 3
- RemoteXY structure with K1/K2/K3 buttons and volume slider
- RTC DS1302 on pins 23, 25, 27
- DHT sensor (pin 28 in final code; pin 9 in earlier diagram - see restoration notes)
- LCD I²C at address 0x27

## Repository Goal

This repository reconstructs the A-SIA project to a clean, documented, and architecturally coherent state. The original codebase was generated by FLProg (a visual PLC-style Arduino IDE) and is preserved as-is in `archived/`. The `src/` directory contains a refactored and modular implementation derived from the same confirmed hardware configuration.

No features have been fabricated. Where evidence was ambiguous or hardware was unconfirmed, this is stated explicitly in `docs/restoration-notes.md`.
